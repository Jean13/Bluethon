Bluethon
Version 0.1
Jean Gonzalez

A Bluetooth Low Energy (BTLE) testing tool. 


To install:
In a terminal, run: python setup.py

To run:
In a terminal, run: python bluethon.py


Functionality:
    * Setup Ubertooth
    * Discover BTLE devices
    * Sniff BTLE traffic
    * Record BTLE messages
    * Decrypt capture files
    * Interact with BTLE devices
    * Fuzz BTLE devices
